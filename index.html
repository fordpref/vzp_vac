vzp_vac

windows vulnerability assessment collection tool

very initial stages...use at own risk.

The tool is designed to be run from the command line without any switches.

To run the tool you need the Sysinternals tools suite in a single directory. The script also needs access to the normal net.exe, sysinfo.exe, netstat.exe, nslookup.exe, ipconfig.exe, nebtstat.exe, and more command line tools in windows. If the system doesn't have access to those, you may want add those (check them on several versions of windows) to the tools directory you create. prepare to have the tools directory in a read only network share if you want to run this script remotely without having to transfer tons of files.

The script will ask you for the location your tools directory and do a simple validation of that path.

The script will then create a local directory on the system being evaluated called c:\assessment and then a sub directory with a date-time-systemname-ip folder.

The script also creates all the frustrating sysinternals EULAAccpeted...so be warned that it is going to create and delete that key, sub keys, and values under HKEY_CURRENT_USER\Software

The script will then run a number of utilities to help give you good assessment data on the system to help you identify items or issues you may have missed with other tools.

It will try to output the data to raw files as well as comma separated files that can be imported into a database.

Soon I hope to have it working with SQLite to go ahead and create that database and have several canned reports to spit out.
